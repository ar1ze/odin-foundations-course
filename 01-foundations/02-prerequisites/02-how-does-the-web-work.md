# How Does the Web Work? 🌐

---

### **Introduction** 💡

To program for the web, you need to understand how the web itself works. Learning these core concepts will give you a holistic understanding of the web ecosystem and help you communicate with other developers.

### **Lesson Overview** 📚

---

* Describe what the internet is.
* Describe what packets are and how they are used to transfer data.
* Understand the differences between a web page, web server, web browser, and search engine.
* Briefly explain what a client is.
* Briefly explain what a server is.
* Explain what IP addresses are.
* Explain what DNS servers are.
